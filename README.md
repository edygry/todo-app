# Todo App

Minimal todo app built with the Skill Builder framework.

## What It Does
- Add tasks
- Mark tasks as done
- Delete tasks
- Clear completed tasks
- Saves to localStorage

## Tech Stack
- Vue 3
- Vite
- localStorage for persistence
